# Organigramast v14.4 — PDF con header corporativo

## Cambio: PDF profesional con branding

El PDF descargado ahora incluye un **header corporativo en la esquina superior izquierda** con:

- **Logo de la caña** Cañaveral (la imagen oficial)
- **"Supertiendas"** en gris claro arriba (peso normal, ~11pt)
- **"Cañaveral"** en verde principal `#1C5A2A` debajo (bold, ~22pt, igual al manual)
- **Línea de acento verde lima** `#7DD105` corriendo bajo todo el header
- **Fecha de generación** en esquina superior derecha (formato es-CO: "30 de abril de 2026")

### Nombre del archivo
Cambia de `organigrama_YYYY-MM-DD.pdf` a `organigrama_canaveral_YYYY-MM-DD.pdf`

### Estructura del PDF
1. Header de 70px (logo + nombre + fecha + línea acento)
2. Padding de 20px alrededor
3. El organigrama dibujado debajo

El branding queda discreto pero presente — se nota empresarial sin invadir el espacio del organigrama.
